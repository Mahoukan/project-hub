Polygon Network

A rotating network of points with colourful connecting lines.

About

This project creates a rotating set of points around a circle and connects every point to every other point, producing changing geometric patterns.

Controls

  • Use the sliders to change speed and number of points
  • Click to add one point
  • Arrow keys adjust the point count
  • Space or R pauses and resumes the animation
  • T toggles trail mode
  • F toggles polygon fill
  • S saves an image
  • H hides or shows the help and sliders
